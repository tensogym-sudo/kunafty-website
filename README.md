# Kunafty Website

A trilingual dessert website for Kunafty, featuring authentic Middle Eastern desserts with French as the default language.

## Features

- **Trilingual Support**: French (default), English, and Arabic
- **Brand Identity**: Custom logo integration
- **TND Pricing**: Tunisian Dinar currency
- **WhatsApp Ordering**: Direct ordering with pre-filled templates
- **Responsive Design**: Mobile-first approach
- **RTL Support**: Proper Arabic text direction

## Tech Stack

- **Next.js 15** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Lucide React** for icons
- **Client Components** with language context

## Pages

- **Home**: Hero section with features and CTAs
- **Menu**: Complete dessert menu with categories
- **Contact**: Location, hours, and contact information

## Getting Started

```bash
npm install
npm run dev
```

The website will be available at `http://localhost:3000` with French as the default language.
